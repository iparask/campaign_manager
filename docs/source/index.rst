Welcome to RADICAL Campaign Manager's documentation!
====================================================
The RADICAL Campaign Manager is a Python library for developing and executing 
computational campaigns. It is being developed by the 
`RADICAL Research Group <http://radical.rutgers.edu/>`_ at Rutgers University. 
The Manager is released under the 
`MIT License <http://opensource.org/licenses/MIT>`_. 


**Project Github Page:** https://github.com/iparask/campaign_manager
